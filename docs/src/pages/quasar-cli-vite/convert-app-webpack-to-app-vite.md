title: Convert q/app-webpack Project
desc: (@quasar/app-vite) How to convert a Quasar CLI with Webpack project to a Quasar CLI with Vite one.
---

This page refers to converting a Quasar CLI with Webpack (`@quasar/app-webpack v4`) project into a Quasar CLI with Vite one (`@quasar/app-vite v3`).

The easiest path is:

- Step 1: Convert your project to [@quasar/app-vite v2](https://legacy-app.quasar.dev/quasar-cli-vite-v2/convert-to-quasar-cli-with-vite) first.
- Step 2: Upgrade to [@quasar/app-vite v3](/quasar-cli-vite/upgrade-guide).
